The Smart Student Planner

I created a clean and responsive web app that helps students manage assignments, deadlines, and productivity.

## 🚀 Features
- Add, delete, and complete tasks
- Deadline tracking (Due Today, Overdue, etc.)
- Automatic task sorting by due date
- Dark mode (saved with localStorage)
- Smooth task completion animations

## 🛠️ Built With
- HTML
- CSS
- JavaScript

## 💡 What I Learned
- DOM manipulation
- Local storage for persistent data
- UI/UX design principles
- Event handling in JavaScript
